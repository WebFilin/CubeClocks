# Cube Clocks

Небольшой проект часов сделанных по принципу управления сегментным табло

- Показывает текущие время
- Помадоро таймер
- Есть возможность выбора даты, времени отсчета для таймера
- Таймер прямого отсчета

## Технологии

- React 17
- React router 6
- Prettier
- Mobx
